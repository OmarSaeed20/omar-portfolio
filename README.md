## рҹ§‘вҖҚрҹ’» Development

Clone the repository, install the dependencies and start the application

```bash
git clone https://github.com/OmarSaeed20/omar-portfolio.git
cd omar-portfolio
npm install
npm run dev
```

**Build for production:**

```bash
npm run build
npm run preview
```

**Note:** use the `depth` parameter to reduce the clone size and speed up the clone.

```sh
git clone --depth=1 https://github.com/OmarSaeed20/omar-portfolio.git
```
